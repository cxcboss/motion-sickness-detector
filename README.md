# 晕不晕 - 驾驶舒适度检测

一个基于手机传感器的驾驶舒适度实时检测工具，用于评估乘车/驾车时的晕车程度。

## 功能特性

- **实时传感器检测**：利用手机加速度计检测急加速、急减速、急转向、顿挫、摇摆等不良事件
- **舒适度评分**：100分制实时评分，分数越低表示越容易晕车
- **晕车等级划分**：参考医学标准，分为无感、轻微、一般、小晕、晕、严重、非常严重7个等级
- **三档灵敏度**：轻度（宽松）、中度（标准）、重度（严格），适应不同评测需求
- **历史报告**：自动保存检测报告，支持查看历史记录
- **深色/浅色主题**：支持手动切换或跟随系统
- **PWA支持**：可添加到手机主屏幕使用

## 晕车等级标准

| 分数范围 | 等级 | 描述 |
|---------|------|------|
| 95-100 | 无感 😊 | 完全没有晕车感 |
| 88-94 | 轻微 🙂 | 几乎感觉不到不适 |
| 78-87 | 一般 😐 | 略有不适但可接受 |
| 65-77 | 小晕 😕 | 开始出现晕车症状 |
| 50-64 | 晕 😣 | 明显晕车不适 |
| 30-49 | 严重 🤮 | 严重晕车，难以忍受 |
| <30 | 非常严重 💀 | 极度不适，建议停车 |

## 检测项目

| 事件类型 | 说明 | 权重 |
|---------|------|------|
| 急减速 | 纵向减速度过大 | ×1.5 |
| 急加速 | 纵向加速度过大 | ×0.75 |
| 急转向 | 横向加速度过大 | ×0.8 |
| 顿挫 | 加速度变化率过大 | ×0.5 |
| 摇摆 | 横向反复摆动 | ×0.5 |

## 灵敏度设置

| 档位 | 说明 | 特点 |
|------|------|------|
| 轻度 | 宽松检测 | 只检测明显不良事件，适合日常驾驶 |
| 中度 | 标准检测 | 平衡灵敏度与准确性，推荐使用 |
| 重度 | 严格检测 | 加减速/顿挫/摇摆参数加强，检测最灵敏 |

## 使用方法

1. 在手机浏览器中打开 `index.html`
2. 点击"授权传感器"授予加速度计权限
3. 将手机平放在仪表台或充电位
4. 点击"开始测评"进入检测
5. 检测过程中保持手机静止
6. 点击"结束检测"查看报告

## 技术栈

- Vue.js 3（Options API）
- DeviceMotion API（加速度传感器）
- Geolocation API（GPS定位）
- localStorage（数据持久化）

## 评分算法

```
舒适度 = 100 - Σ(事件等级 × 事件等级 × 权重)
```

事件等级：1（轻微）、2（中等）、3（严重）
